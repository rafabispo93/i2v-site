Quick Start

Download

Download the I2V to have access to all resources.

Download I2V

Installation

Step One

In order to use the I2V is necessary to download and import the dependencies used by I2V. The dependencies are shown bellow:

Leaflet 0.7

https://leafletjs.com/2015/09/01/leaflet-0.7.4-released.html

Leaflet Heatmap

https://www.patrick-wied.at/static/heatmapjs/plugin-leaflet-layer.html

Leaflet Indoor

https://cdn.jsdelivr.net/gh/cbaines/leaflet-indoor/leaflet-indoor.js

Step Two

As second step you have to import the downloaded I2V files .

Import the I2V CSS

rel="stylesheet" href="css/i2v.css"

Import the I2V JS

src="js/i2v.js"

Step Three

Put The imgs directory into your web page structure in the same i2v.js tree structure.

        var cfg = {
            // radius should be small ONLY if scaleRadius is true (or small radius is intended)
            // if scaleRadius is false it will be the constant radius used in pixels
            "radius": 0.00006,
            "maxOpacity": .8,
            "z-index": 10000,
            // scales the radius based on map zoom
            "scaleRadius": true,
            // if set to false the heatmap uses the global maximum for colorization
            // if activated: uses the data maximum within the current map boundaries
            //   (there will always be a red spot with useLocalExtremas true)
            "useLocalExtrema": true,
            // which field name in your data represents the latitude - default "lat"
            latField: 'lat',
            // which field name in your data represents the longitude - default "lng"
            lngField: 'lng',
            // which field name in your data represents the data value - default "value"
            valueField: 'count'
        };

        var osmUrl = 'https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png',

                maxZoom: 22,
                attribution: "Map data © OpenStreetMap contributors"
            });
        var map = new L.Map('map', {
            layers: [osm],
            center: new L.LatLng(49.41873, 8.67689),
            // center: new L.LatLng(-12.9633078, -38.5005932),
            zoom: 19
        });
        var i2v = new i2v();
        $(document).ready(function(){

          var req = new XMLHttpRequest();
          req.open("GET", "data.json");
          req.overrideMimeType("application/json");
          req.send(null);
          req.onreadystatechange = function () {
              if (req.readyState == 4 && req.status == 200) {
                  i2v.generatePlant(JSON.parse(req.responseText)); // callback
              }
          }



          map.on('click', function(e){
              var coord = e.latlng;
              var lat = coord.lat;
              var lng = coord.lng;
              console.log("latitude: " + lat + " longitude: " + lng);
          });


        });




    </script>
